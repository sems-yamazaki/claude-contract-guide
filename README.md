# Claude 契約判断ガイド

Claude（Anthropic）の法人利用における **API契約 / アプリ契約** の選定基準を、判定ツール付きで整理した社内検討用の単一HTML資料。

## 公開ページ

GitHub Pages: https://sems-yamazaki.github.io/claude-contract-guide/

## 内容

- §01 用語集（API / アプリ / Self-serve / Enterprise / Tier / SCIM 等）
- §02 契約ルート判定ツール（API・アプリ別、各8問のインタラクティブ判定）
- §03 契約の2階建て構造
- §04 アプリ契約の詳細（Team / Enterprise self-serve / sales-assisted）
- §05 API契約の詳細（Self-serve / Enterprise、Tier制度、商用組込の規約）
- §06 次のアクション（契約フロー）

各セクション末尾に一次情報・二次情報の出典URLを明記。

## 注意

- 料金・割引率・リードタイム等は作成時点（2026-06）の公開情報および集計値に基づく目安。最新情報は Anthropic 公式（claude.com/pricing 等）で要確認。
- 為替レートは 1 USD = 150 JPY で換算。
